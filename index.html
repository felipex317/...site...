Quem somos

Somos uma pequena empresa que temos o principal foco em Marketing e jogos, Relizamos programação de sites em html,css, java script e php!

Nossos objetivos

Nosso objetivo, é dar uma melhor visibilidade a iniciantes de projetos WEB, seja com a divulgação de um estabelecimento ou uma minima linha de código.

Ferramentas Que Utilizamos

Html

HTML é uma linguagem usada para criar páginas na web. Com ela, você pode adicionar texto, imagens e links, organizando tudo de forma que os navegadores possam entender e mostrar aos usuários.

CSS

CSS (Cascading Style Sheets) é uma linguagem usada para estilizar e dar estilo às páginas da web criadas com HTML. Com o CSS, você pode controlar a aparência do texto, como tamanho, cor, fonte e espaçamento, permitindo criar páginas visualmente atraentes e consistentes.

JavaScript

JavaScript é uma linguagem de programação usada para tornar as páginas da web interativas. Com ele, é possível adicionar funcionalidades como animações, validações de formulários e interação com o usuário.

PhP

PHP é uma linguagem de programação para criar páginas da web dinâmicas. Com PHP, você pode processar formulários, acessar bancos de dados e gerar conteúdo interativo. É frequentemente usado em conjunto com HTML para criar sites que se adaptam às necessidades dos usuários.
